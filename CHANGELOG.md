# Changelog

## 0.1.5

Released: 2026-04-02

No breaking changes.

### Added

- broad multi-language coverage across workout list, detail, active sessions, completion, history, editor, and settings
- a language selector and structured settings layout for the webview app

### Changed

- the workout editor is redesigned with grouped cards, clearer field labels, and direct exercise delete actions
- workout badges, previews, timers, and list rows now use tighter translated copy throughout the app

### Notes

- existing workouts, session history, and local storage data remain compatible with this release
